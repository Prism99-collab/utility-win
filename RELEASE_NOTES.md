# Utility Win 1.0.0

First public release of Utility Win, a fast Spotlight-style launcher for Windows.

## Highlights

- Global hotkey launcher with configurable shortcuts.
- Premium rounded dark UI with focused search on open.
- Search for Start Menu apps, curated quick actions, aliases, and user files.
- Smarter file category detection for folders, PDFs, Office files, images, audio, video, archives, code, and data files.
- Real shell icons for apps and files where available.
- Settings for hotkey, startup behavior, search sources, result count, aliases, and system-action confirmations.
- Single-instance tray app with reload, settings, show, and quit actions.

## Installer

Download and run the MSI from the release assets. The installer creates a desktop shortcut named **Utility Win**.
